# Generated Guides (Drafts)

These Markdown files are **draft guides** generated from the author’s Facebook group posts export (`group_posts_and_comments.html`).

Important:
- These drafts are **not** integrated into the live website.
- Each guide is stored in a folder with **four language versions**: `es`, `fr`, `de`, `en`.
- Structure is kept consistent across languages to support later publishing.
