# Arquitectura del sitio

## Portal: espanolesensuiza.ch

---

# 1. Estructura general

El sitio web debe organizarse de forma jerárquica para facilitar la navegación y el SEO.

Estructura principal:

/
├── vivir-en-suiza
├── trabajo
├── tramites
├── sistema-suizo
└── actualidad

---

# 2. Página principal

Contenido:

- introducción
- acceso rápido a las secciones
- artículos destacados
- recursos oficiales
- acceso a la comunidad

---

# 3. Estructura de navegación

El sitio debe incluir:

- menú principal
- enlaces internos entre artículos
- navegación clara entre secciones

---

# 4. Organización del repositorio

El repositorio se organiza de la siguiente manera:

/
├── index.html
├── style.css
├── app.js
├── sitemap.xml
├── robots.txt
├── media/
├── docs/

---

# 5. Escalabilidad

La arquitectura debe permitir:

- añadir nuevas secciones
- añadir nuevos artículos
- ampliar funcionalidades
